# Tiny JS

A tiny JavaScript implementation built for learning, with a focus on understanding how the language works under the hood.

## 💻 About the project

This project was created to study JavaScript at a deeper level by rebuilding core pieces from scratch.

In practice, Tiny JS is a simplified JavaScript-like language for educational purposes. The goal is not to to build a clear and practical foundation for learning language internals.

I am using this project to explore topics like:

- How source code is read, tokenized, and interpreted
- How to model language behavior in small, testable components
- How language rules become data structures and runtime behavior

## 🎨 Implemented features

The scope is intentionally incremental, so features are added step by step.

Right now, Tiny JS is focused on:

- Initial TypeScript project setup
- First lexer and tokenization milestones
- Test foundation for future language behavior

This project is being developed with TypeScript.

## 🎯 What I am learning

Even as a simplified implementation, building a language has been a great way to understand concepts that are usually hidden when we only consume JavaScript APIs.

Main learning points so far:

- How lexical analysis and interpretation connect
- How to represent tokens and language rules internally
